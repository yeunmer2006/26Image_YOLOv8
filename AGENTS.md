# 项目规范

## 要求

Plan 模式不许修改，只给规划，不许写代码

## 项目背景

图像处理与识别大作业：YOLOv8 目标检测模型微调与性能验证
团队4人，协同完成

## 技术规范

### 环境要求

- Python 3.8+
- PyTorch >= 1.8
- Ultralytics YOLOv8

### 编码规范

- 缩进：Tab
- 变量命名：snake_case（Python）
- 函数命名：动词开头，描述性强
- 代码注释：中文详细说明

### 分支管理

- main: 主分支，稳定版本
- dev: 开发分支
- feature/: 功能分支

### 提交规范

```
<类型>: <简短描述>

可选的详细说明
```

类型：
- feat: 新功能
- fix: 修复
- docs: 文档
- test: 测试
- refactor: 重构

## 任务检查清单

### 必做项

- [ ] 选择 YOLOv8 模型变体（n/s/m/l/x）
- [ ] 选择检测数据集（禁止 COCO）
- [ ] 按官方格式准备数据集
- [ ] 完成模型微调训练
- [ ] 记录微调参数和设置
- [ ] 对比微调前后性能指标
- [ ] 个人完成额外5张图片测试

### 提交物

- 微调过程详细记录
- 性能比较报告
- 5张图片测试结果与分析

## 注意事项

1. 数据集不能选择 COCO
2. 每位学生需独立完成额外验证部分
3. 性能测试需记录 mAP、precision、recall 等指标
4. 保留所有训练日志和可视化结果
