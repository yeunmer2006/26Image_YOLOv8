## 阶段1：环境确认（第10周）

### 1.1 检查当前配置
```bash
cat config.py
```

确认关键配置：
```python
MODEL_NAME = 'yolov8n.pt'           # ✓ 确认
DATA_YAML = 'SKU-110K.yaml'         # ✓ 确认
MOSAIC = 0.0                         # ✓ 确认（关闭Mosaic）
SAVE_PERIOD = 5                      # ✓ 确认（每5个Epoch保存）
```

### 1.2 安装依赖
```bash
pip install ultralytics
pip list | grep ultralytics
```

### 1.3 确认GPU可用
```bash
python -c "import torch; print(f'CUDA: {torch.cuda.is_available()}, Device: {torch.cuda.get_device_name(0)}')"
```

### 1.4 测试代码是否可以运行（不下载数据）

在下载数据集之前，先测试代码逻辑是否正确：

```bash
# 测试 1：导入检查
python -c "from ultralytics import YOLO; import config; print('导入成功')"

# 测试 2：模型加载检查（只加载不下训练）
python -c "from ultralytics import YOLO; m = YOLO('yolov8n.pt'); print(f'模型加载成功: {m.model_name}')"

# 测试 3：配置检查
python -c "import config; print(f'MODEL_NAME: {config.MODEL_NAME}'); print(f'MOSAIC: {config.MOSAIC}'); print(f'SAVE_PERIOD: {config.SAVE_PERIOD}')"

# 测试 4：训练脚本语法检查
python -m py_compile train.py && echo "train.py 语法正确"

# 测试 5：配置文件语法检查
python -m py_compile config.py && echo "config.py 语法正确"

# 测试 6：验证 train.py 中的 mosaic 参数
grep -n "mosaic" train.py
```

**预期输出**：
- 所有导入成功
- 模型加载成功
- 配置值正确（MOSAIC=0.0, SAVE_PERIOD=5）
- 语法检查通过
- train.py 中包含 `mosaic=config.MOSAIC`

**如果测试失败**，先修复再继续：
- 依赖缺失 → `pip install ultralytics`
- 模型加载失败 → 检查网络连接
- 配置值错误 → 检查 config.py

## 阶段3：模型训练（第11-12周）

### 3.1 训练命令
```bash
python run.py
```

### 3.2 训练参数（来自 config.py）
| 参数 | 值 |
|------|-----|
| 模型 | yolov8n.pt |
| Epochs | 100 |
| 图像尺寸 | 640 |
| Batch Size | 16 |
| Mosaic | 0.0（关闭） |
| 学习率 | 0.01 |
| 早停耐心 | 50 |

### 3.3 保存策略
- 每 5 个 Epoch 保存一次 checkpoint
- 保存位置：`yolo_finetune/homeobjects_n_no_mosaic/`
- 文件：`weights/epoch5.pt`, `epoch10.pt`, ..., `epoch100.pt`

### 3.4 训练监控指标
训练过程中记录：
- box_loss（边界框损失）
- cls_loss（分类损失）
- dfl_loss（分布式焦点损失）
- mAP@0.5
- mAP@0.5:0.95
- Precision
- Recall
